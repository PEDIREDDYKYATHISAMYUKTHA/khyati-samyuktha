    About This Project

    This system detects human face shapes using deep learning and recommends suitable hairstyles. It also includes a Missing Person Detection module that scans entire CCTV or mobile videos and identifies the reference person with accuracy percentage.

    Technologies Used

    • Python & Flask
    • OpenCV
    • MediaPipe Face Mesh (468 Landmarks)
    • Face Recognition Library (dlib)
    • Deep Learning CNN Model (TensorFlow / Keras)
    • HTML, CSS, JavaScript

    Features

    • ✅ Face shape detection (Heart, Oval, Round, Square, Oblong)
    • ✅ Hairstyle recommendation based on face shape
    • ✅ Live webcam face detection
    • ✅ Scans ENTIRE video at once
    • ✅ Only detects reference person — ignores all other faces
    • ✅ Blue box + blue tick ✓ on matched person
    • ✅ Accuracy % shown on best matched frame
    • ✅ Progress bar while scanning
    • ✅ Alert sound when person is found
